# Архив промптов

Эта директория содержит предыдущие версии промптов для различных задач системы.

## Формат именования
- summary_v1.txt - первая версия промпта для саммари
- summary_v2.txt - вторая версия и т.д.

## Структура промпта
Каждый промпт должен содержать:
1. Четкое описание роли ассистента
2. Формат вывода
3. Ограничения (длина, стиль, язык)
4. Контекст задачи
